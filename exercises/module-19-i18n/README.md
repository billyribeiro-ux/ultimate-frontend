# Module 19 — Internationalization (i18n): Exercises

These five exercises build on the core concepts from Module 19: message extraction, ICU message formatting, locale-based routing, RTL layout support, and SEO with hreflang tags.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Message Extraction | beginner | 10 min | Message catalogs, key-based lookups, translation files |
| 2 | ICU Plural Rules | intermediate | 20 min | ICU MessageFormat, pluralization, interpolation |
| 3 | Locale Routing | advanced | 30 min | URL-based locale, param matching, layout inheritance |
| 4 | RTL Layout | expert | 45 min | Logical properties, dir attribute, bidirectional text |
| 5 | Hreflang SEO | principal | 60 min | Alternate links, x-default, sitemap integration, canonical URLs |

## Prerequisites

- Lessons 19.1 through 19.7 completed
- Understanding of SvelteKit routing and layouts
- `pnpm dev` running successfully

## How to Use

1. Read the exercise brief carefully
2. Attempt the solution without hints first
3. Use progressive hints only when genuinely stuck
4. Compare your solution to the provided one — multiple approaches are valid
5. Each exercise builds confidence for the next; do them in order

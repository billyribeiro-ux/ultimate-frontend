# Module 20 — Testing: Exercises

These five exercises build on the core concepts from Module 20: Vitest configuration, unit testing stores, component render testing, Playwright end-to-end testing, and CI pipeline configuration.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Vitest Configuration | beginner | 10 min | vitest.config.ts, test runner setup, first test |
| 2 | Store Unit Test | intermediate | 20 min | Testing .svelte.ts reactive classes, assertion patterns |
| 3 | Component Render Test | advanced | 30 min | @testing-library/svelte, render, user events, queries |
| 4 | Playwright Locator | expert | 45 min | E2E selectors, page interactions, assertions, fixtures |
| 5 | CI Pipeline Config | principal | 60 min | GitHub Actions, test matrix, artifact management |

## Prerequisites

- Lessons 20.1 through 20.7 completed
- Vitest and Playwright installed
- `pnpm dev` running successfully

## How to Use

1. Read the exercise brief carefully
2. Attempt the solution without hints first
3. Use progressive hints only when genuinely stuck
4. Compare your solution to the provided one — multiple approaches are valid
5. Each exercise builds confidence for the next; do them in order

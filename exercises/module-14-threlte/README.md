# Module 14 — Threlte (3D in Svelte): Exercises

These five exercises build on the core concepts from Module 14: integrating Three.js via Threlte in SvelteKit, creating 3D scenes, loading GLTF models, handling mesh interactions, animating cameras, and optimizing 3D for production.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Your First 3D Scene | beginner | 10 min | Canvas setup, T.Mesh, T.BoxGeometry, lighting |
| 2 | GLTF Model Loader | intermediate | 20 min | useGltf, Suspense, loading states, model positioning |
| 3 | Interactive Mesh | advanced | 30 min | Pointer events on meshes, hover/click state, material swaps |
| 4 | Scroll-Driven Camera | expert | 45 min | useTask, scroll progress, camera interpolation, easing |
| 5 | Production 3D with Lazy Loading | principal | 60 min | Dynamic imports, LOD strategy, performance budget, mobile fallback |

## Prerequisites

- Lessons 14.1 through 14.7 completed
- `@threlte/core` and `@threlte/extras` installed
- Familiarity with Three.js coordinate system basics
- `pnpm dev` running successfully

## How to Use

1. Read the exercise brief carefully
2. Attempt the solution without hints first
3. Use progressive hints only when genuinely stuck
4. Compare your solution to the provided one — multiple approaches are valid
5. Each exercise builds confidence for the next; do them in order

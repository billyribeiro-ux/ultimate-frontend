# Module 1 — Foundation: Exercises

These five exercises build on the core concepts from Module 1: creating Svelte components, TypeScript strict typing, the three-block file structure, PE7 CSS architecture, fluid typography, and scoped styles.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Profile Card from Scratch | beginner | 10 min | Component structure, template expressions, scoped styles |
| 2 | Typed Contact Form | intermediate | 20 min | TypeScript interfaces, PE7 tokens, fluid spacing |
| 3 | Prove Scoping Works | advanced | 30 min | CSS scoping mechanics, DevTools verification, specificity |
| 4 | Fluid Pricing Table | expert | 45 min | clamp(), OKLCH, CSS Grid, TypeScript generics |
| 5 | Design Token Audit Tool | principal | 60 min | Token architecture, type system design, mobile-first |

## Prerequisites

- Lessons 1.1 through 1.9 completed
- `pnpm dev` running successfully
- Familiarity with PE7 token system in `src/app.css`

## How to Use

1. Read the exercise brief carefully
2. Attempt the solution without hints first
3. Use progressive hints only when genuinely stuck
4. Compare your solution to the provided one — multiple approaches are valid
5. Each exercise builds confidence for the next; do them in order

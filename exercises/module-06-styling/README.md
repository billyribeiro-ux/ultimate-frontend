# Module 6 — Styling: Exercises

These five exercises build on Module 6's core concepts: @layer architecture, OKLCH color theory, CSS Grid/Flexbox, container queries, transitions, springs, and reduced-motion accessibility.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Layer Specificity Challenge | beginner | 10 min | @layer ordering, cascade control, specificity |
| 2 | OKLCH Palette Generator | intermediate | 20 min | OKLCH color math, perceptual uniformity, dynamic palettes |
| 3 | Grid Without Breakpoints | advanced | 30 min | CSS Grid auto-fit/fill, intrinsic sizing, no media queries |
| 4 | Animation Sequence | expert | 45 min | CSS transitions, stagger, Svelte transitions, choreography |
| 5 | Reduced-Motion Audit | principal | 60 min | Accessibility audit, alternative experiences, user preferences |

## Prerequisites

- Module 1-5 exercises completed
- Lessons 6.1 through 6.18 completed
- Understanding of CSS cascade and specificity

# Module 4 — Control Flow: Exercises

These five exercises build on Module 4's core concepts: `{#if}`, `{#each}` with keys, nested iteration, `{#key}`, `{#await}`, and async/await patterns.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Conditional Notification Banner | beginner | 10 min | {#if}/{:else if}/{:else}, conditional classes |
| 2 | Keyed List Reorder | intermediate | 20 min | {#each} with keys, array manipulation, animation |
| 3 | Nested Data Tree | advanced | 30 min | Recursive {#each}, nested iteration, tree structures |
| 4 | Async Error States | expert | 45 min | {#await}, error boundaries, retry logic, loading states |
| 5 | Typed Fetch Pipeline | principal | 60 min | Generic fetch wrapper, streaming, type narrowing, caching |

## Prerequisites

- Module 1-3 exercises completed
- Lessons 4.1 through 4.10 completed
- Understanding of JavaScript Promises and async/await

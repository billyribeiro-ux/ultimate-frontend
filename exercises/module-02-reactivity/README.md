# Module 2 — Reactivity: Exercises

These five exercises build on Module 2's core concepts: `$state`, `$derived`, `$effect`, reactive objects/arrays, `$state.raw`, `$state.snapshot`, and effect cleanup.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Counter Variants | beginner | 10 min | $state with primitives, event handlers |
| 2 | Derived Chain Calculator | intermediate | 20 min | $derived, chained derivations, TypeScript |
| 3 | Effect Cleanup Puzzle | advanced | 30 min | $effect, cleanup functions, memory leaks |
| 4 | Snapshot Serialization | expert | 45 min | $state.snapshot, $state.raw, JSON serialization |
| 5 | Finite State Machine | principal | 60 min | State machine design, reactive classes, transitions |

## Prerequisites

- Module 1 exercises completed
- Lessons 2.1 through 2.13 completed
- Understanding of JavaScript closures and references

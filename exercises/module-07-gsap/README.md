# Module 7 — GSAP & Animation: Exercises

These five exercises build on Module 7's core concepts: GSAP timelines, ScrollTrigger, Svelte actions, transition coexistence, and Threlte 3D.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Timeline Choreography | beginner | 10 min | gsap.timeline, sequencing, basic tweens |
| 2 | ScrollTrigger Coordination | intermediate | 20 min | ScrollTrigger, pin, scrub, progress tracking |
| 3 | Action-Based Animation | advanced | 30 min | use: directive, reusable animation actions, cleanup |
| 4 | GSAP + Svelte Transitions | expert | 45 min | Coexistence strategy, lifecycle coordination, hand-off |
| 5 | Threlte 3D Scene | principal | 60 min | Three.js via Threlte, 3D in Svelte, performance |

## Prerequisites

- Module 1-6 exercises completed
- Lessons 7.1 through 7.14 completed
- GSAP installed via pnpm

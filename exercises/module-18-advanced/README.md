# Module 18 — Advanced Patterns: Exercises

These five exercises build on the core concepts from Module 18: compound components, polymorphic rendering, state machines, custom preprocessors, and monorepo architecture.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Compound Component | beginner | 10 min | Context API, component composition, slot-based API |
| 2 | Polymorphic `as` Prop | intermediate | 20 min | Dynamic element rendering, type-safe props, HTML semantics |
| 3 | State Machine | advanced | 30 min | Finite state machines, transition guards, event-driven UI |
| 4 | Custom Preprocessor | expert | 45 min | Svelte preprocessor API, AST manipulation, build pipeline |
| 5 | Monorepo Setup | principal | 60 min | Workspace configuration, shared packages, dependency management |

## Prerequisites

- Lessons 18.1 through 18.7 completed
- Strong understanding of Svelte 5 runes and component patterns
- `pnpm dev` running successfully

## How to Use

1. Read the exercise brief carefully
2. Attempt the solution without hints first
3. Use progressive hints only when genuinely stuck
4. Compare your solution to the provided one — multiple approaches are valid
5. Each exercise builds confidence for the next; do them in order

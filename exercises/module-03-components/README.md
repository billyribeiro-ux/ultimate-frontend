# Module 3 — Components: Exercises

These five exercises build on Module 3's core concepts: component extraction, typed props with `$props()`, `$bindable`, snippets, composition patterns, and container queries.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Extract and Type a Card | beginner | 10 min | $props(), interface Props, basic extraction |
| 2 | Bindable Toggle Switch | intermediate | 20 min | $bindable, two-way binding, accessibility |
| 3 | Snippet Render-Prop Pattern | advanced | 30 min | {#snippet}, {@render}, typed snippet props |
| 4 | Compound Component Set | expert | 45 min | Composition, context bridging, container queries |
| 5 | Headless Combobox | principal | 60 min | Render delegation, keyboard nav, ARIA, generics |

## Prerequisites

- Module 1 and 2 exercises completed
- Lessons 3.1 through 3.10 completed
- Understanding of TypeScript generics

# Module 16 — Database: Exercises

These five exercises build on the core concepts from Module 16: defining database schemas, performing CRUD operations, querying relations, managing migrations, and working with transactions in SvelteKit.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Schema Definition | beginner | 10 min | Table definition, column types, TypeScript inference |
| 2 | Insert and Select | intermediate | 20 min | Insert operations, select queries, type-safe results |
| 3 | Relations Query | advanced | 30 min | Foreign keys, joins, nested data retrieval |
| 4 | Migration Workflow | expert | 45 min | Schema evolution, migration files, rollback strategy |
| 5 | Transaction Safety | principal | 60 min | Atomic operations, rollback on failure, consistency guarantees |

## Prerequisites

- Lessons 16.1 through 16.7 completed
- Database ORM/query builder configured (e.g., Drizzle ORM)
- `pnpm dev` running successfully

## How to Use

1. Read the exercise brief carefully
2. Attempt the solution without hints first
3. Use progressive hints only when genuinely stuck
4. Compare your solution to the provided one — multiple approaches are valid
5. Each exercise builds confidence for the next; do them in order

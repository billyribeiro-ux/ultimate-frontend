# Module 15 — Authentication: Exercises

These five exercises build on the core concepts from Module 15: session management, user registration, route protection, logout flows, and OAuth integration in SvelteKit.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Session Check Guard | beginner | 10 min | Hooks, locals, session reading |
| 2 | Registration Form | intermediate | 20 min | Form actions, validation, password hashing |
| 3 | Protected Route Redirect | advanced | 30 min | Load functions, redirects, auth state |
| 4 | Logout Flow | expert | 45 min | Cookie clearing, invalidation, redirect chains |
| 5 | OAuth Mock Provider | principal | 60 min | OAuth flow, callback handling, token exchange |

## Prerequisites

- Lessons 15.1 through 15.7 completed
- Understanding of SvelteKit hooks and form actions
- `pnpm dev` running successfully

## How to Use

1. Read the exercise brief carefully
2. Attempt the solution without hints first
3. Use progressive hints only when genuinely stuck
4. Compare your solution to the provided one — multiple approaches are valid
5. Each exercise builds confidence for the next; do them in order

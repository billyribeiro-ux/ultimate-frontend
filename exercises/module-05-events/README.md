# Module 5 — Events: Exercises

These five exercises build on Module 5's core concepts: lowercase event attributes, typed handlers, preventDefault/stopPropagation, event forwarding via callbacks, closures, debouncing/throttling, and accessible interactions.

## Difficulty Progression

| # | Title | Difficulty | Time | Key Skills |
|---|-------|-----------|------|------------|
| 1 | Click Handler Patterns | beginner | 10 min | onclick, inline handlers, TypeScript MouseEvent |
| 2 | Typed Event Forwarding | intermediate | 20 min | Callback props, custom event contracts, parent-child comm |
| 3 | Debounce Implementation | advanced | 30 min | Debounce from scratch, closures, effect cleanup |
| 4 | Closure Puzzle | expert | 45 min | Stale closure bugs, event handler identity, array closures |
| 5 | Keyboard-Accessible Widget | principal | 60 min | ARIA patterns, roving tabindex, keyboard navigation |

## Prerequisites

- Module 1-4 exercises completed
- Lessons 5.1 through 5.10 completed
- Understanding of closures and function references

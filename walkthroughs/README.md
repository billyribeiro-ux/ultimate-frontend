# Walkthroughs — Build Real Apps from Scratch

These are **end-to-end project walkthroughs** that guide you through building a complete application from `pnpm create svelte` to deployment. They are not module lessons — they are guided tutorials that pull together concepts from many modules into a single cohesive build.

Each walkthrough includes complete code (no "fill in the rest yourself"), architecture decisions with explanations, file trees at each milestone, and a deployment section at the end.

---

## Available Walkthroughs

### 1. [SaaS Analytics Dashboard](./saas-dashboard.md)

**Time:** ~10 hours | **Difficulty:** Advanced | **Modules referenced:** M2, M7, M9A, M9B, M11, M12, M13, M15, M16, M17, M22

Build a full SaaS analytics dashboard with authentication, a Drizzle database, live-updating metric widgets powered by remote functions and SSE, a sortable/filterable event log with TanStack Table, GSAP-animated counters, error boundaries, SEO for marketing pages, and deployment to Vercel Edge.

### 2. [E-Commerce Store](./ecommerce-store.md)

**Time:** ~10 hours | **Difficulty:** Intermediate-Advanced | **Modules referenced:** M2, M4, M6, M8, M9A, M10, M11, M12, M13, M15

Build a complete e-commerce store with product listings, dynamic product detail pages, a reactive shopping cart, checkout with Valibot validation, SSG for product pages, SSR for search, JSON-LD structured data, optimized images, and OKLCH-branded product cards with container queries.

### 3. [Developer Portfolio](./developer-portfolio.md)

**Time:** ~8 hours | **Difficulty:** Intermediate | **Modules referenced:** M1, M6, M7, M9A, M10, M12, M13, M14, M19

Build a developer portfolio with a PE7 OKLCH color palette, GSAP scroll animations, a statically generated blog, a progressively enhanced contact form, a 3D Threlte hero, full SEO with Open Graph, Lighthouse 100 optimization, and English/Portuguese internationalization.

### 4. [Blog Platform](./blog-platform.md)

**Time:** ~8 hours | **Difficulty:** Intermediate-Advanced | **Modules referenced:** M2, M9A, M9B, M10, M13, M15, M16, M18

Build a full blog platform with Markdown rendering, an authenticated admin panel, CRUD for posts via form actions, a Drizzle database for posts/tags/authors, a comment system with remote functions, a dynamic sitemap and RSS feed, prerendered post pages, and full-text search.

### 5. [Real-Time Chat App](./realtime-chat.md)

**Time:** ~8 hours | **Difficulty:** Advanced | **Modules referenced:** M2, M5, M11, M12, M15, M16, M17

Build a real-time chat application with WebSocket connection management, session-based auth, message history from a database, optimistic message sending, typing indicators via SSE, user presence tracking, keyboard accessibility, message reactions, and scroll anchoring.
